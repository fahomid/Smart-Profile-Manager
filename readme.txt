# Project Title

Smart Profile Manager

## Getting Started

Simply install the apk file into you mobile phone from \SmartProfileManager\app\build\outputs\apk folder. The main root folder contains all the project files.

### Prerequisites

Simply use "Android Studio" to import the project and then use it as you need.
